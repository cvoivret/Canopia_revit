Pour installer addin canopia :
1) copier le manifeste CANOPIA_GUI.addin dans le dossier "C:\Users\xxx\AppData\Roaming\Autodesk\Revit\Addins\2022". Remplacer xxx par le nom d'user
2) copier les deux dll canopia-* au même endroit.
3) configurer la variable "Assembly" dans le manifest pour pointer vers les dll. Le chemin va probablement être de la forme "C:\Users\xxx\AppData\Roaming\Autodesk\Revit\Addins\2022"
4) démarrer ou re-démarrer REVIT et accepter le chargement
5) un onglet CANOPIA est normalement accessible
